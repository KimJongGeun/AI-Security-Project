# AI Security Study

AI 보안을 학습하고 직접 구현해보는 공간입니다.

## 목표

- AI/LLM 보안 위협 이해 및 대응 방안 학습
- 실습 기반으로 공격/방어 시나리오 구현
- 보안 도구 및 프레임워크 활용
